# Finance Data Analyzer

A Python tool for analyzing and visualizing stock market data using real-time financial data.

## Features
- Pull live stock data for any ticker via Yahoo Finance
- Calculate 20-day and 50-day moving averages
- Compute annualized volatility
- RSI (Relative Strength Index) with overbought/oversold signals
- Normalize and compare multiple stocks on one chart
- Export analyzed data to CSV

## Technologies
- Python, Pandas, Matplotlib, yfinance

## Usage
```bash
python Project.py
```
Select option 1 to analyze a single stock or option 2 to compare multiple stocks.

## Example Output
- Price chart with moving averages
- Volatility over time
- RSI indicator
- CSV export of all metrics
